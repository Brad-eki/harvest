Harvest
==============

Harvest allows its users to record time spent on tasks. Tracking time can allow for increased productivity, as businesses better understand what business practices lead to wasted time. 

Requirements
==============
- Python 2.7
- Django 1.6.2
- Bootstrap 3.1.1
